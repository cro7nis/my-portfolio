A selection of my ML and web3 related repos.

Llama3 assistant hosted on Akash Network

Llama 3 assistant powerered by Ollama and Streamlit. Special thanks to Meta for releasing Llama 3. The assistant uses the 8B-parameter model which is fast and requires ~5GB VRAM and it can even run on CPU with reasonable speed. The project is developed for the "Create and deploy guide" zealy task which is hosted by Akash.

Text-to-image model hosted on Akash Network

Text-to-image app powerered by black-forest-labs /FLUX.1-schnell. The project is developed for the Flux-schnell model on Akash zealy task which is hosted by Akash. Instead of using the replicate api this project hosts the model from hugging face directly.

Subtitle Generator hosted on Akash Network

Automatic subtitle generator using faster whisper repo. Faster whisper is a remplementation of OpenAI's Whisper using CTranslate2 for faster inference. The Subtible generator consists of an API that is built with FastAPI and a UI powered by Streamlit.
